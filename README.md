# print-test

Mini projet web statique pour tester l'impression PDF.

## Fichiers

- `index.html` : interface (Pico CSS)
- `app.js` : logique JS (selection du billet, journal, appel `POST /print`)

## Utilisation

Ouvrir `index.html` dans un navigateur, puis cliquer sur **Imprimer le PDF**.
Le backend d'impression est attendu sur `http://localhost:8080/print`.
